An image is non-text visual content that needs an appropriate text alternative unless it is purely decorative.

### Semantic markup

*   The image's role is included in the accessibility tree.
    
    *   **Type:** WCAG
        
    *   **Success Criteria:** 1.1.1 Non-text Content
        
*   The image's text alternative is included as the accessible name in the accessibility tree.
    
    *   **Type:** WCAG
        
    *   **Success Criteria:** 1.1.1 Non-text Content
        
*   **If the image is decorative:** The image is purely decorative. It is not included in the accessibility tree.
    
    *   **Type:** WCAG
        
    *   **Success Criteria:** 1.1.1 Non-text Content
        
### Media alternatives

*   The image has a text alternative that communicates the meaning or content that is displayed visually
    
    *   **Type:** WCAG
        
    *   **Success Criteria:** 1.1.1 Non-text Content
        
### Visual design

*   The image is not an image of text, unless the image is a logo, the presentation of text is essential to convey the information or the presentation can be visually customised to meet the user's needs.
    
    *   **Type:** WCAG
        
    *   **Success Criteria:** 1.4.5 Images of Text
