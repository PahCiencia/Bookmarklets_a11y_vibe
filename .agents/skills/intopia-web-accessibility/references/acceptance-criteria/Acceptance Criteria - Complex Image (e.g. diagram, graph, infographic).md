A complex image is a visual such as a diagram, chart, or infographic that requires a detailed text description to convey its meaning.

### Semantic markup

*   The complex image's simple text alternative is included as the accessible name in the accessibility tree.
    
    *   **Type:** WCAG
        
    *   **Success Criteria:** 1.1.1 Non-text Content
        
*   The complex image's role is included in the accessibility tree.
    
    *   **Type:** WCAG
        
    *   **Success Criteria:** 1.1.1 Non-text Content
        
### Media alternatives

*   The complex image has a detailed text description that communicates the essential information displayed visually.
    
    *   **Type:** WCAG
        
    *   **Success Criteria:** 1.1.1 Non-text Content
        
*   The complex image has a simple text alternative that identifies the image, and can indicate the location of the detailed text description.
    
    *   **Type:** WCAG
        
    *   **Success Criteria:** 1.1.1 Non-text Content
        
### Visual design

*   The detailed text description is located on the same page close to the image or on another page but is easily discoverable from the image.
    
    *   **Type:** WCAG
        
    *   **Success Criteria:** 1.1.1 Non-text Content
        
*   Graphs or diagrams don’t rely on colour alone to convey information.
    
    *   **Type:** WCAG
        
    *   **Success Criteria:** 1.4.1 Use of Color
