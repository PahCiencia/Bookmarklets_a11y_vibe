A heading is a structural label that introduces and organizes sections of content in a logical hierarchy.

### Labels and messaging

*   The H1 describes the overall topic of the page.
    
    *   **Type:** WCAG
        
    *   **Success Criteria:** 2.4.6 Headings and Labels
        
*   The H1 is the same or similar to the page title.
    
    *   **Type:** Best Practice
        
*   The H1 is positioned at the start of the main content.
    
    *   **Type:** Best Practice
        
*   There is only one H1 on the page.
    
    *   **Type:** Best Practice
        
*   The section headings concisely describe the topic or purpose of the content that follows them.
    
    *   **Type:** WCAG
        
    *   **Success Criteria:** 2.4.6 Headings and Labels
        
### Semantic markup

*   The heading role and level are included in the accessibility tree.
    
    *   **Type:** WCAG
        
    *   **Success Criteria:** 1.3.1 Info and Relationships
        
*   The heading follows a logical hierarchy; for example, H3 is used for subheadings under H2.
    
    *   **Type:** WCAG
        
    *   **Success Criteria:** 1.3.1 Info and Relationships
        
*   Heading levels do not skip; for example, the sequence should not jump from H2 directly to H4.
    
    *   **Type:** Best Practice
        
### Visual design

*   The heading text meets the minimum contrast requirement of 4.5:1 against the background colour, or 3:1 for large-scale text (24 px or 18.5px and bold).
    
    *   **Type:** WCAG
        
    *   **Success Criteria:** 1.4.3 Contrast (Minimum)
        
### **Adaptive UI**

*   The heading can be increased in size up to 200% without truncating, overlapping with other text or being cut off.
    
    *   **Type:** WCAG
        
    *   **Success Criteria:** 1.4.4 Resize Text
        
*   Text spacing can be increased without text overlapping, truncating or being cut off. See 1.4.12 Text Spacing for exact requirements.
    
    *   **Type:** WCAG
        
    *   **Success Criteria**: 1.4.12 Text Spacing
