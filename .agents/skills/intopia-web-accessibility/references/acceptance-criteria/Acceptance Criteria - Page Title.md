A page title is the text shown in the browser title bar or tab that identifies the current page.

### Labels and messaging

*   The page title accurately describes the content of the page.
    
    *   **Type:** WCAG
        
    *   **Success Criteria:** 2.4.2 Page Titled (A)
        
*   The page title is unique across the website.
    
    *   **Type:** Best Practice
        
*   The page title prioritises unique, page-specific information before the general site name.
    
    *   **Type:** Best Practice
        
### Semantic markup

*   The page has a title element (`<title>`) in the `<head>`.
    
    *   **Type:** WCAG
        
    *   **Success Criteria:** 2.4.2 Page Titled (A)
