Abre una página, haz clic en el bookmarklet y obtén resultados visuales inmediatos junto con un informe JSON listo para compartir con un LLM.

Ver instrucciones

Cómo usar

  1. 01
    Arrastra el bookmarklet que quieras a tu barra de favoritos del navegador.
  2. 02
    Navega a cualquier página web que quieras analizar.
  3. 03
    Haz clic en el bookmarklet. Aparecerá el panel lateral con los resultados.
  4. 04
    Pulsa "Copiar JSON" para obtener el informe completo y pegarlo en un LLM.

Herramientas

Arrastra cualquier bookmarklet a tu barra de favoritos para instalarlo.

Resultado JSON para LLM

Cada bookmarklet devuelve un objeto JSON con este esquema. Cópialo y pégalo en tu LLM favorito para obtener recomendaciones de corrección.
