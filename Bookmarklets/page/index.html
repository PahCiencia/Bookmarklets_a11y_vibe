Auditoría de Accesibilidad Web

8 bookmarklets para testear y mejorar la accesibilidad de páginas web. Análisis visual + JSON para LLMs.

WCAG 2.2 JSON Output Sin dependencias Accesible

¿Qué son estos bookmarklets?

Son pequeños scripts que instalas en tu navegador para analizar la accesibilidad de cualquier página web en tiempo real. Obtén resultados visuales en la página + JSON estructurado para procesar con LLMs.

✨ Características principales

🎨

Panel Visual

Resultados en un panel flotante accesible sin interferir con la página

📊

JSON Normalizado

Exporta resultados en JSON para procesar con IA/LLMs con 1 click

📚

Recomendaciones WCAG

Cada problema incluye recomendación específica para arreglarlo

⚡

Instantáneo

Ejecuta en segundos, sin carga de dependencias pesadas

📚 Instalación en 4 pasos

1

Copia el código

Elige un bookmarklet abajo y haz clic en "Copiar Bookmarklet". Se copiará automáticamente al portapapeles.

2

Abre marcadores de tu navegador

Abre la barra de marcadores:

Chrome/Edge: Ctrl+Shift+B (Cmd+Shift+B en Mac) Firefox: Ctrl+Shift+B Safari: Cmd+Option+B
3

Crea un nuevo marcador

Click derecho en la barra → "Agregar página" o "Nuevo marcador"

Nombre: "A11y - Headings" (o similar)

URL: Pega el código que copiaste en el paso 1

4

¡Úsalo!

Navega a cualquier página web y haz clic en tu nuevo bookmarklet. ¡Aparecerá el panel de análisis!

Los 8 Bookmarklets

Core (Esenciales)

📋

Analizar Headings

Estructura de títulos

Valida la estructura jerárquica de headings (h1-h6). Detecta h1 único, saltos de nivel, headings vacíos.

  • H1 único detectado
  • Saltos de nivel (H1→H3)
  • Headings vacíos
  • Orden jerárquico
🖼️

Analizar Imágenes

Textos alternativos

Detecta imágenes sin alt, alt vacío o genérico. Valida la calidad y relevancia del texto alternativo.

  • Alt presente/ausente
  • Alt genérico o débil
  • Nombres de archivo
  • Contexto visual
🎨

Analizar Contraste

Ratios de color WCAG

Valida el contraste entre texto y fondo. Verifica cumplimiento WCAG AA (4.5:1) y AAA (7:1).

  • Ratios de contraste
  • Cumplimiento AA/AAA
  • Texto grande detectado
  • Elementos resaltados
⚙️

Auditoría Axe-Core

Análisis automático

Ejecuta análisis automático completo usando axe-core. Detecta 50+ violaciones WCAG 2.2 AA.

  • 50+ reglas automáticas
  • Violaciones WCAG
  • Problemas incompletos
  • Elementos que pasan

Adicionales (Especializados)

📝

Análisis de Texto Alt

Calidad profunda

Análisis profundo de la calidad del texto alt en imágenes. Puntuación 0-100 por imagen con recomendaciones.

  • Puntuación 0-100
  • Patrones débiles
  • Longitud óptima
  • Sugerencias mejora
🏗️

Análisis Semántica HTML

Estructura correcta

Valida el uso de elementos semánticos: nav, main, section, article, footer. Detecta divitis y estructura incorrecta.

  • Elementos semánticos
  • Main único
  • Estructura lógica
  • Atributo lang
📋

Análisis de Formularios

Labels y acceso

Valida que las labels estén asociadas correctamente a inputs. Verifica fieldsets, legends, y accesibilidad general.

  • Labels asociadas
  • Relación for/id
  • Fieldsets con legend
  • Campos accesibles
🎨

Ratios de Color WCAG

Análisis detallado

Análisis detallado y profesional de ratios de contraste. Tabla comparativa, breakdown AA/AAA, elementos problemáticos.

  • Ratios precisos
  • Tabla comparativa
  • WCAG AA/AAA
  • Elementos fallidos

🤖 Usar con LLMs

Flujo típico

  1. Ejecutas bookmarklet en la página web que quieres auditar
  2. Haces click en "Copiar JSON"
  3. Pegas el JSON en tu LLM favorito (ChatGPT, Claude, etc.)
  4. LLM genera código HTML/CSS corregido basado en los problemas
  5. Implementas cambios en tu proyecto
